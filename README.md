# Game List Search App

Web application that displays a list of games with fuzzy search support.

## Tech Stack
- React.js
- Ruby on Rails
- PostgreSQL

## Features
- Game listing via /list API
- Fuzzy search via /list?search=
- Responsive UI based on provided design
